# 묵상만개 앱 개인정보 처리방침 (Privacy Policy)

최종 업데이트: 2025년 11월 19일

묵상만개(이하 “앱”)는 사용자의 개인정보를 수집하지 않으며,
사용자의 어떠한 개인 정보도 외부로 전송하거나 저장하지 않습니다.

## 1. 수집하는 개인정보

- 본 앱은 아래와 같이 어떠한 개인정보도 수집하지 않습니다.

- 이름, 이메일, 전화번호 등 개인 식별 정보

- 위치 정보

- 사진/영상/파일

- 광고 ID, 디바이스 식별자 등 추적 정보

- 사용 기록, 로그 데이터

- 앱에서 처리되는 모든 정보는 기기(사용자 단말기) 내에서만 일시적으로 사용되며, 외부 서버로 전송되지 않습니다.

## 2. 개인정보 저장 및 보관

본 앱은

서버에 데이터를 저장하지 않으며

사용 기록을 클라우드나 제3자 서비스로 보내지 않으며

분석 도구(Analytics)를 사용하지 않습니다.

앱 내에서 일시적으로 처리되는 데이터는 사용자가 앱을 종료하면 즉시 삭제되거나 더 이상 접근할 수 없습니다.

## 3. 제3자 제공

본 앱은 개인정보를 어떠한 제3자에게도 제공하지 않습니다.

## 4. 외부 서비스 연동

본 앱은 다음과 같은 외부 서비스와 어떠한 연동도 하지 않습니다.

- 광고 플랫폼

- 소셜 로그인

- 외부 서버 API

- 통계/분석 서비스

즉, 앱은 완전히 독립적으로 동작합니다.

## 5. 어린이 개인정보 보호

본 앱은 13세 미만 아동을 대상으로 개인정보를 수집하지 않습니다.
수집하지 않기 때문에 별도의 어린이 보호 정책이 불필요합니다.

## 6. 개인정보 보호를 위한 기술적 조치

앱은 개인정보를 저장하거나 전송하지 않지만,
사용자 데이터가 외부로 유출되지 않도록 아래 원칙을 따릅니다.

- 입력된 데이터는 네트워크를 통해 전송되지 않음

- 기기 내에서만 일시 처리

- 지속적인 저장 없음

## 7. 개인정보 처리방침 변경

본 정책은 앱 기능 변경 또는 관련 법령 개정에 따라 업데이트될 수 있습니다.
변경사항은 앱 업데이트 노트 또는 본 페이지를 통해 안내됩니다.
